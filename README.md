## Задание 2
Создать функцию **isCorrectRect(rect)**.

### Аргументы:
- `rect` — список из двух кортежей:
  1. Первый кортеж — координаты левого нижнего угла `(x1, y1)`.
  2. Второй кортеж — координаты верхнего правого угла `(x2, y2)`.

### Возвращает:
- `True`, если прямоугольник корректный.
- `False`, если некорректный.

## Задание 3
Создать функцию **isCollisionRect(rect1, rect2)**.

### Аргументы:
- `rect1`, `rect2` — списки из двух кортежей (углы прямоугольников).

### Возвращает:
- `True`, если прямоугольники пересекаются.
- `False`, если не пересекаются.

### Ошибки:
- Если прямоугольник некорректный, вызвать **RectCorrectError** с комментарием `"1й прямоугольник некоректный"`.

## Задание 4
Создать функцию **intersectionAreaRect(rect1, rect2)**.

### Аргументы:
- `rect1`, `rect2` — списки из двух кортежей:
  1. Первый кортеж — координаты левого нижнего угла прямоугольника `(x1, y1)`.
  2. Второй кортеж — координаты верхнего правого угла прямоугольника `(x2, y2)`.

### Возвращает:
- Площадь пересечения двух прямоугольников.
- `0`, если пересечения нет.

### Ошибки:
- Если прямоугольник некорректный, функция должна вызвать **ValueError** с соответствующим комментарием.

## Задание 5
Создать функцию **intersectionAreaMultiRect(rects)**.

### Аргументы:
- `rects` — список списков, каждый из которых состоит из двух кортежей:
  1. Первый кортеж — координаты левого нижнего угла прямоугольника `(x1, y1)`.
  2. Второй кортеж — координаты верхнего правого угла прямоугольника `(x2, y2)`.

### Возвращает:
- Общую площадь пересечения всех переданных прямоугольников.
- Учитываются только уникальные области пересечения без повторений.

### Ошибки:
- Если в функцию передан некорректный прямоугольник, функция должна вызвать **RectCorrectError** с соответствующим комментарием.
